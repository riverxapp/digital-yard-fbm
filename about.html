Chirag Dodiya

Senior Software Engineer

Chirag is a Senior Software Engineer with 8+ years of experience designing and delivering production-grade web and cloud-native applications. He specializes in building scalable backend services, performant front-end interfaces, and reliable CI/CD pipelines. He enjoys mentoring engineers, improving delivery practices, and solving hard architecture problems.

Primary focus: Distributed systems, backend engineering (API design, microservices), frontend engineering (React, modern JavaScript), and cloud platforms (AWS, Kubernetes).

Experience Summary

  • 8+ years in full-stack and backend engineering across startups and enterprise teams.
  • Led development of high-throughput services serving millions of requests per day.
  • Designed and migrated services to cloud-native infrastructure using Kubernetes and managed cloud services.
  • Built observable, testable systems with strong emphasis on reliability and performance.

Core Strengths

  • System Design: Scalable architecture, service boundaries, event-driven patterns.
  • Backend: APIs, microservices, performance tuning (Java/Python/Node).
  • Frontend: Modern JS, React, component-driven design and UX considerations.
  • Cloud & DevOps: Kubernetes, AWS, CI/CD, observability (Prometheus, ELK).
  • Leadership: Mentorship, code reviews, cross-functional collaboration.
